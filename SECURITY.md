# Security Policy
## Reporting a Vulnerability
Please do NOT open a public issue. Report securely via:
1. **GitHub Security Advisory**: Open a draft advisory at https://github.com/Kockerpro94/repository-security-baseline/security/advisories.
We will acknowledge your report within 48 hours.
## Secure Coding Practices
- Mandatory input validation
- Parameterized queries
- Secure handling of secrets
- Regular dependency audits
