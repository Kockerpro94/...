# Security Policy

## Supported Versions

Security updates are provided for the latest version of this project.

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please do not open a public issue. Instead, please report it through the GitHub Security Advisory system for this repository:

https://github.com/Kockerpro94/repository-security-baseline/security/advisories

All security reports will be promptly acknowledged and addressed.
