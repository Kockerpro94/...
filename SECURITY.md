# Security Policy

## Supported Versions

Only the latest version of this project is supported with security updates.

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please report it by opening a **Draft Security Advisory** on GitHub.

Please include:
- A detailed description of the vulnerability
- Steps to reproduce the issue
- Potential impact

We will review your report and respond as soon as possible. Thank you for helping keep this project secure!
