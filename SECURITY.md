# Security Policy

## Supported Versions

Only the latest version of this project is supported for security updates.

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please report it to us by opening a draft security advisory on GitHub.

You should receive a response within 48 hours. If you do not receive a response, please follow up on the advisory.

Please provide a detailed description of the vulnerability, including steps to reproduce it if possible.

Thank you for helping us keep this project secure!
