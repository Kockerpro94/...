# Security Policy

## Supported Versions

Only the latest version of this project is supported for security updates.

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please report it by opening a draft security advisory on GitHub.

We commit to the following:
- Acknowledge receipt of your report within 48 hours.
- Provide an estimated timeline for resolution.
- Notify you once the vulnerability is fixed.

Please do not report security vulnerabilities through public GitHub issues.
